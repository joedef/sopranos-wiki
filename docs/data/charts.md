## Building your own

Copy this and set the widths by hand — there's no runtime calculation, so check your arithmetic. Add `alt` or `warn` to the bar class for the blue or gold variants:

```html
<div class="chart" markdown>

<div class="chart-row">
<div class="chart-label">Label</div>
<div class="chart-track"><div class="chart-bar" style="width:72%"></div></div>
<div class="chart-value">72</div>
</div>

Caption: what the reader should take from it, and where the number came from.
{ .chart-caption }

</div>
```

Widths are proportions of the largest value in that chart. Always caption with the source. Never plot a number the underlying data page doesn't support.
